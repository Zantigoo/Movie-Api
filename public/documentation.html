Flixir API

A movie finding machine

An API for retrieving information on movies

The Following are the current API endpoints and their basic function.
  • '/documentation' - Lists documentation for myFlix API, isn't that how you got here?
  • '/genres', '/movies', '/directors'
    GET will respond with each entry in the given database, presented in JSON.
  • '/movies/(title)', '/genres/(name)' and '/directors/(name)' - GET will respond with a specific entry in the chosen database, also in JSON.
  • '/users' - POST will register a new user. Body must be a JSON in provided format.
  • '/users/(username) - DELETE will remove that specified user from the database. Can also recieve a PUT with a compatible JSON file to modify profiles.
  • '/users/movies/(objectID)' - POST will add a new favorite movie to that users favorites list'
    DELETE will remove it instead.

User endpoints use a JSON in the proper format. Template provided here.

{ "Username": String,
"Password": String,
"Email": String,
}
