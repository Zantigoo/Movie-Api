myFlix Api Instructions

A movie finding machine

An API for retrieving information on movies

The Following are the current API endpoints and their basic function.
  • '/documentation' - Lists documentation for myFlix API, isn't that how you got here?
  • '/genre', '/movies', '/directors'
    GET will respond with each entry in the given database, presented in JSON.
  • '/movies/(name)', '/genre/(name)' and '/directors/(name)' - GET will respond with a specific entry in the chosen database, also in JSON.
  • '/user' - POST will register a new user. Body must be a JSON with a "name" and "password" specified, also generates an uuid for user
  • '/user/(id) - DELETE will remove that specified user from the database.
  • '/user/change/(current name)/(desired name)' - PUT will change the specified user to the desired username.
  • '/user/favorite/(name)' - POST will add a new favorite movie to that users favorites list'
    DELETE will do the same but removing it instead.
